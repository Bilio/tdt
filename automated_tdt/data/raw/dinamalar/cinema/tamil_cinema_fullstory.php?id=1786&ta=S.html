மேலும் விமர்சனம்

சர்தார் கப்பார் சிங் (தெலுங்கு)
ஆந்திர தேசமே எதிர்பார்த்து காத்திருந்த பவன் கல்யாணின் 'சர்தார் கப்பார் சிங்' படம் மிகப்பெரும் பொருட்செலவில் தயாராகி வந்தது. ...
முதல் தகவல் அறிக்கை
வெள்ளித்திரை டாக்கீஸ் எனும் பேனரில் ஏ.ஆர்.முஜீப் தயாரிக்க, விலாசம் படத்தின் இயக்குனர் பா.ராஜகணேசனின் எழுத்து, ...
ஜேக்கப்பிண்டே சுவர்க்க ராஜ்யம் (மலையாளம்)
நடிகர்கள் : நிவின்பாலி, ரெபா மோனிகா ஜான், ரெஞ்சி பணிக்கர், லட்சுமி ராமகிருஷ்ணன், வினீத் சீனிவாசன், அஜூ வர்கீஸ் இசை : ஷான் ரஹ்மான்ஒளிப்பதிவு : ...
ஜித்தன்-2
வின்சென்ட் செல்வாவின் கதை, திரைக்கதை, இயக்கத்தில் சுமார் பத்து ஆண்டுகளுக்கு முன் வெளிவந்த ஜித்தனின் ...
ஓய்
இளையராஜா இசையில், நீண்ட இடைவெளிக்குப் பின் சங்கிலி முருகன் தலைமையில், புதியவர் பிரான்சிஸ் மார்க்ஸ் ...
